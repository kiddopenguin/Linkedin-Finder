# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2026-02-17

### Adicionado
- Dashboard responsivo com Bootstrap 5
- Upload de arquivos Excel (.xlsx, .xls, .csv)
- Configuração de URLs personalizadas com checkboxes
- Detecção automática de colunas do Excel
- Suporte para nome e sobrenome separados
- Concatenação automática de nome completo
- Geração de arquivo Excel apenas com URLs
- Processamento via AJAX
- Validação de arquivos (tipo e tamanho)
- Limpeza automática de arquivos temporários
- Preview em tempo real da URL gerada
- Mapeamento inteligente de colunas (PT/EN)

### Recursos
- Variáveis suportadas: Nome, Empresa, Cargo, Localização, Email, Telefone
- Valores fixos ou dinâmicos do Excel
- Interface intuitiva e moderna
- Feedback visual durante processamento
